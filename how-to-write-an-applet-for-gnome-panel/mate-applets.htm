mate-applets

 Как написать Applet для панели
 
MATE-Applets is a fork of GNOME-Applets

Applets for use with the MATE panel
    https://github.com/mate-desktop/mate-applets

You need at least:
  mate-panel
  gtk+-2.24.0
  glib-2.36.0
  libgtop-2.12.0
  libwnck

Some extras that improve functionality:
  libnotify
  dbus-0.34
